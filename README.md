# Telecom - churn-analysis-dashboard
End-to-end customer churn analysis using SQL, Python ML, and Power BI
![Dashboard](Telecom - churn analysis dashboard.png)

📌 Project Overview

This project analyzes customer churn for a telecom company using **SQL Server**, **Python (Machine Learning)**, and **Power BI**. The dataset consists of **7,043 customers** with detailed demographic, service usage, billing, and churn information.

### 🎯 Objective
- Identify patterns that differentiate churned customers from retained customers
- Understand the main drivers of churn using SQL analysis and Machine Learning
- Build predictive models to score every customer with a churn risk probability
- Create an interactive Power BI dashboard for business stakeholders
- Recommend actionable strategies to reduce churn and improve retention

# 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| **SQL Server** | Data exploration, segmentation, views for Power BI |
| **Python** | Data preprocessing, ML pipeline, predictions export |
| **scikit-learn** | Random Forest, Logistic Regression, Gradient Boosting |
| **pandas / NumPy** | Data manipulation and analysis |
| **Power BI** | Interactive dashboard, KPIs, drill-through filtering |
| **DAX** | Calculated measures and columns |
